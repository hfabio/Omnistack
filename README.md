## Getting Started

This project was created with Node.JS, ReactJS and React-native. Back-end uses nodemon, so you just need to ```yarn dev``` inside the folder to run the server.

### Prerequisites

```
NodeJS
NPM
Yarn
react-app
react-native-app
```
